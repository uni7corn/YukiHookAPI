# 更新日志

> 这里记录了 `YukiHookAPI` 的版本更新历史。

::: danger

我们只会对最新的 API 版本进行维护，若你正在使用过时的 API 版本则代表你自愿放弃一切维护的可能性。

:::

### 1.2.1 | 2024.06.20 &ensp;<Badge type="tip" text="最新" vertical="middle" />

- 捕获单例 Hooker 中的异常，防止其阻断整个进程
